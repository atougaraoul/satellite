About OrbitWatch

OrbitWatch uses Two-Line Elements (TLE) and SGP4 propagation (via satellite.js) to compute satellite positions and display them on a map. TLEs are public data available from sources like Celestrak.

How it works

  1. Fetch TLE for a satellite (by NORAD catalog number).
  2. Use SGP4 to compute Earth-centered inertial (ECI) coordinates at the current time.
  3. Convert to geodetic coordinates (lat/long/alt) and plot on the map.

If you want real-time telemetry from operators (telemetry streams) or integration with N2YO, Space-Track, or a ground-station API, I can add those integrations.
